## CSS 변수 정의

```css
:root {
  /* ───────────────────────────────
     Primary Colors (웜 로즈 베이지)
  ─────────────────────────────── */
  --color-primary:       #D4B5A8;   /* 메인 포인트 — 스킨톤 베이지 로즈 */
  --color-primary-light: #EAD8D0;   /* 호버 밝기, 배지 배경 */
  --color-primary-dark:  #B8927F;   /* 눌림 상태, 강조 테두리 */

  /* ───────────────────────────────
     Secondary Colors (딥 차콜)
  ─────────────────────────────── */
  --color-secondary:     #1A1A1A;   /* 브랜드명, 헤딩, 주요 버튼 */
  --color-accent:        #C08B7A;   /* 더스티 로즈 — 컬러 스와치, 포인트 */

  /* ───────────────────────────────
     Background Colors
  ─────────────────────────────── */
  --color-bg-primary:    #FFFFFF;   /* 메인 콘텐츠 배경 */
  --color-bg-secondary:  #F5F3F1;   /* 섹션 구분 배경 (오프화이트) */
  --color-bg-tertiary:   #F0EBE7;   /* 카드, 모달 배경 (라이트 베이지) */
  --color-bg-banner:     #E8746A;   /* 공지 배너 (코랄 레드) */

  /* ───────────────────────────────
     Text Colors
  ─────────────────────────────── */
  --color-text-primary:  #1A1A1A;   /* 제목, 브랜드명, 주요 본문 */
  --color-text-secondary:#4A4A4A;   /* 부제목, 설명 텍스트 */
  --color-text-muted:    #8A8A8A;   /* 캡션, 플레이스홀더, 보조 정보 */

  /* ───────────────────────────────
     Border / Outline
  ─────────────────────────────── */
  --color-border-light:  #E8E2DD;   /* 섹션 구분선, 인풋 기본 테두리 */
  --color-border-mid:    #D4CCC7;   /* 카드 테두리, 구분 강조선 */

  /* ───────────────────────────────
     Interactive (버튼, 링크)
  ─────────────────────────────── */
  --color-button-primary: #1A1A1A;  /* 기본 CTA 버튼 배경 */
  --color-button-hover:   #3A3A3A;  /* 버튼 호버 상태 */
  --color-button-outline: #D4B5A8;  /* 아웃라인 버튼 테두리/텍스트 */
  --color-link:           #C08B7A;  /* 링크 기본 색상 */
  --color-link-hover:     #B8927F;  /* 링크 호버 색상 */

  /* ───────────────────────────────
     Shadow / Overlay
  ─────────────────────────────── */
  --color-shadow:         rgba(26, 26, 26, 0.08);   /* 카드 그림자 */
  --color-overlay:        rgba(26, 26, 26, 0.45);   /* 모달 딤 오버레이 */
}
```

## 컬러 사용 가이드

### Primary Color — `#D4B5A8` (웜 베이지 로즈)
| 사용 상황 | 적용 예시 |
|-----------|-----------|
| 섹션 헤딩 강조 밑줄 | `border-bottom: 2px solid var(--color-primary)` |
| 태그, 배지 배경 | 스킬 태그, 카테고리 레이블 |
| 아웃라인 버튼 | "더 보기", "자세히" 보조 버튼 |
| 이미지 플레이스홀더 | 로딩 중 스켈레톤 배경 |

```css
/* 사용 예시 — 강조 태그 */
.tag {
  background-color: var(--color-primary-light);
  color: var(--color-primary-dark);
  border: 1px solid var(--color-primary);
  border-radius: 20px;
  padding: 4px 12px;
}
```

---

### Secondary Color — `#1A1A1A` (딥 차콜 블랙)
| 사용 상황 | 적용 예시 |
|-----------|-----------|
| 주요 헤딩 (H1~H3) | 프로젝트 제목, 섹션명 |
| 네비게이션 텍스트 | 메뉴 항목, 로고 |
| 기본 CTA 버튼 | "프로젝트 보기", "연락하기" |
| 아이콘 | SVG 아이콘 fill 색상 |

```css
/* 사용 예시 — CTA 버튼 */
.btn-primary {
  background-color: var(--color-secondary);
  color: var(--color-bg-primary);
  transition: background-color 0.2s ease;
}
.btn-primary:hover {
  background-color: var(--color-button-hover);
}
```

---

### Accent Color — `#C08B7A` (더스티 로즈)
| 사용 상황 | 적용 예시 |
|-----------|-----------|
| 링크 텍스트 | 인라인 링크, 네비게이션 활성 상태 |
| 호버 포인트 | 카드 호버 시 테두리 변색 |
| 진행 바 | 스킬 레벨 표시바 |
| 아이콘 강조 | 소셜 링크 호버 |

```css
/* 사용 예시 — 카드 호버 효과 */
.project-card {
  border: 1px solid var(--color-border-light);
  transition: border-color 0.2s ease, box-shadow 0.2s ease;
}
.project-card:hover {
  border-color: var(--color-accent);
  box-shadow: 0 4px 20px var(--color-shadow);
}
```

### Background 계층 구조
```
Level 0 — #FFFFFF  : 최상단 콘텐츠 카드, 모달
Level 1 — #F5F3F1  : 페이지 메인 배경
Level 2 — #F0EBE7  : 섹션 교차 배경, 사이드바
```

```css
/* 짝수/홀수 섹션 교차 배경 패턴 */
section:nth-child(odd)  { background-color: var(--color-bg-primary); }
section:nth-child(even) { background-color: var(--color-bg-secondary); }
```

---

### 텍스트 계층 구조
| 변수 | 색상 | 사용처 |
|------|------|--------|
| `--color-text-primary` | `#1A1A1A` | H1~H3, 강조 본문 |
| `--color-text-secondary` | `#4A4A4A` | 일반 본문, 부제목 |
| `--color-text-muted` | `#8A8A8A` | 날짜, 캡션, 메타 정보 |

```css
h1, h2, h3 { color: var(--color-text-primary); }
p, li       { color: var(--color-text-secondary); }
time, small { color: var(--color-text-muted); }
```

## 반응형 고려사항

### 다크 모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경 반전 — 딥 뉴트럴 계열 유지 */
    --color-bg-primary:    #1C1917;   /* 웜 다크 (순수 블랙 지양) */
    --color-bg-secondary:  #242220;
    --color-bg-tertiary:   #2C2927;

    /* 텍스트 반전 */
    --color-text-primary:  #F0EBE7;
    --color-text-secondary:#C8BDB8;
    --color-text-muted:    #8A8A8A;   /* 그레이는 유지 */

    /* Primary/Accent — 채도 약간 낮춰 눈 부심 방지 */
    --color-primary:       #C8A898;
    --color-accent:        #B07868;

    /* 테두리 — 어두운 배경에 맞게 조정 */
    --color-border-light:  #3A3532;
    --color-border-mid:    #4A4542;

    /* 버튼 — 라이트 반전 */
    --color-button-primary: #F0EBE7;
    --color-button-hover:   #FFFFFF;

    /* 그림자 — 다크 모드에서 더 진하게 */
    --color-shadow:         rgba(0, 0, 0, 0.30);
    --color-overlay:        rgba(0, 0, 0, 0.65);
  }
}
```

### 수동 다크 모드 토글 (JavaScript 연동)

```css
/* data-theme 속성으로 수동 제어 */
[data-theme="dark"] {
  --color-bg-primary:     #1C1917;
  --color-bg-secondary:   #242220;
  --color-text-primary:   #F0EBE7;
  --color-text-secondary: #C8BDB8;
  --color-border-light:   #3A3532;
  /* ... 위 다크 모드 값과 동일 적용 */
}
```

```javascript
// 토글 예시
const toggle = () => {
  const theme = document.documentElement.getAttribute('data-theme');
  document.documentElement.setAttribute(
    'data-theme',
    theme === 'dark' ? 'light' : 'dark'
  );
};
```
