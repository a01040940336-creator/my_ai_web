# lecture1 프로젝트 설정

## 역할
이 디렉토리는 React + MUI 학습 실습 공간입니다.

## 문서 연결
@docs/design-system.md
@docs/code-convention.md
@docs/new_project.md

## 개발 환경
- Vite + React 18
- Material-UI (MUI) v9
- React Router DOM v7
- Roboto 폰트

## 주요 규칙
- 컴포넌트는 `src/components/` 폴더에 배치
- 페이지는 `src/pages/` 폴더에 배치
- MUI 컴포넌트를 우선 사용
- 디자인 시스템 문서의 색상 팔레트 준수

## 새 프로젝트 생성
`_template_settings` 폴더를 복사하여 빠르게 시작 가능
